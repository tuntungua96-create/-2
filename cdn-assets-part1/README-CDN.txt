CDN assets package for the visual UI site.

Upload the contents of this folder to your CDN while keeping the folder structure.

After upload, send Codex the CDN base URL, for example:
https://cdn.example.com/pointe-assets

Codex will then rewrite local references to:
https://cdn.example.com/pointe-assets/主站/fonts/OPlusSans3-Regular.woff2
https://cdn.example.com/pointe-assets/music/bg.mp3
https://cdn.example.com/pointe-assets/我的项目/images/1/grid.avif
